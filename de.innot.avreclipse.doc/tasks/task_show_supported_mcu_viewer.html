Using the AVR Supported MCUs View

The AVR Supported MCUs view is an information tool. It shows information about the MCUs supported by the currently active toolchain. This can be used for new projects to select a MCU that is compatible with all tools. Also this view allows access to the processor datasheets.

To open the view select Window > Show view... > other...

Select AVR Supported MCUs under the the AVR entry and click OK.

The view is now shown:

 

This view shows a list of all MCUs known to the plugin and whether or not they

  • are supported by the different tools of the active toolchain (avr-gcc and avrdude columns)
  • are included in the Atmel Part Description Files (AVR Studio column)
  • have fuse and/or lockbit definitions (Fuses column)

The Signature column is just for info because there was still some space left in the table. It could be checked to see which MCUs are hardware compatible (like the at90usb1286 and
at90usb1287, which have the same signature 0x1E9782)

Those MCUs for which datasheets links are known to the plugin are displayed as hyperlinks. Clicking on them will download and open the datasheet. This list is currently static and will only be updated together with new versions of the plugin. Future versions of the plugin may include an editor to add other links.
